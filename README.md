# 🧩 検察パズル組立ゲーム

検察官になって証拠を組み立て、事件を立証するパズルゲームです。制限時間内に適切な判断を下しましょう。

## 🎮 ゲームの概要

バラバラになった証拠のピースを正しく配置して事件を立証するゲームです。すべてのピースが揃えば「公判請求」、揃わないと判断したら「不起訴」を選択します。ただし、適正な捜査を尽くさなければゲームオーバーになります。

## 🎯 ゲームルール

### 基本操作
- **左側の配置エリア**: 薄い文字で表示された証拠に対応するピースを配置
- **右側のピースエリア**: 1〜12の証拠からランダムに9個が表示される（4×3グリッド）
- **ドラッグ&ドロップ**: 右のピースを左の対応する枠にドラッグして配置
- **クリックで戻す**: 配置したピースをクリックすると元に戻せる

### 証拠の種類（1〜12）
1. 目撃者
2. 防犯カメラ
3. 凶器
4. 指紋
5. GPS
6. 鑑定書
7. 通話履歴
8. SNS
9. 供述調書
10. 実況見分
11. 秘密の暴露
12. 被害届

### 制限時間
- **各パズル30秒以内**に判断する必要があります
- 残り5秒未満になると赤字で警告されます
- タイムアップでゲームオーバー

### 判定方法

#### ✅ 公判請求（成功）
- すべてのピースを正しく配置すると自動的に公判請求となります
- 次のステージに進みます

#### ❌ 不起訴
- 証拠が揃わないと判断したら「不起訴」ボタンを押します
- **重要**: 最低10秒は捜査を尽くす必要があります
  - 10秒未満で不起訴を押すとゲームオーバー
- まだ配置できるピースがあるのに不起訴を押すとゲームオーバー

#### 🔍 補充捜査
- 未使用のピースをランダムに1〜3個変更できます
- 捜査を尽くすための重要な手段です
- ペナルティはありません

### ⚠️ ゲームオーバー条件
以下のいずれかに該当するとゲームオーバーです：

1. **時間切れ**: 30秒以内に判断できなかった
2. **捜査不十分**: 10秒未満で不起訴を選択した
3. **配置可能な証拠の見落とし**: まだ配置できるピースがあるのに不起訴を選択した

ゲームオーバー時は「適正な捜査を心がけましょう。」というメッセージが表示され、最初からやり直しになります。

## 📊 スコアリング

- 経過時間がスコアに加算されます（少ないほど良い）
- 不正解の判断をするとペナルティが加算されます

## 🎲 ゲーム性のポイント

### パズルの完成可能性
- 右側に表示される9個のピースが、左側に必要な9個の証拠と一致する場合のみ完成可能
- 完成不可能なパズルも出現するため、適切に見極める必要がある

### 戦略的な判断
1. **初期確認**: まず右側のピースと左側の必要証拠を確認
2. **配置可能チェック**: 一致するピースを優先的に配置
3. **補充捜査の活用**: 足りないピースがある場合は補充捜査を試す
4. **不起訴の判断**: 10秒以上捜査し、配置できるピースがないことを確認してから不起訴

## 🚀 プレイ方法

### ローカル環境で起動
1. `index.html` をブラウザで開く
2. タイトル画面で「ゲームスタート」ボタンをクリック
3. ゲーム開始
